| Роль  | Права роли | Группы пользователей |
| --- | --- | --- |
| Название роли, которое отвечает требованиям RBAC в Kubernets. | Укажите права, которые необходимо выдать этой роли. | Выделите группы пользователей в организации, которые нужно связать с этой ролью. |
| secret-reader | Просмотр секретов | DevOps-инженера, Инженеры по эксплуатации, Специалист по ИБ |
| devops-access | Управление кластером | DevOps-инженера, Инженеры по эксплуатации, Специалист по ИБ |
| view | Только просмотр ресурсов кластера. Без возможности модификации. | Владелец продукта, бизнес-аналитики, менеджеры операционной команды. |
| dev-access | модификация ресурсов в назначенных namespaces, но без доступа к секретам или кластер-левел ресурсам. | Разработчики, часть функциональной команды. |
